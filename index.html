Interactive Visualizations of Working Memory and Music Data

Team SEABORN!

Team Members: Nicolas Toon, Sophia Vo, Simone Rao

Chosen Dataset: A Multimodal Dataset for Investigating Working Memory in Presence of Music

Aspects we plan to use:

  • Reaction Time Data
  • Session Time Data
  • Correct Response Data
  • Data grouped by task complexity and/or session

Question: How does the type of background music and task complexity influence the memory of certain letters?
